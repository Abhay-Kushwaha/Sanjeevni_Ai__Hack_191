    ## ğŸ” Workflow

    ### For Patients:
    1. **Disease Prediction**
    - Upload MRI (Brain Tumor / Alzheimer's)
    - Or select symptoms (Diabetes, Kidney, Heart, etc.)
    2. **Get Results**

    - Book medicines
